# Правила проекта

- Проект независим от основного Vlogme.
- Утверждённый стиль не менять.
- Не проводить редизайн без прямого задания.
- Не изменять глобальные CSS-переменные без отдельного задания.
- Не менять шапку, навигацию и подвал без крайней необходимости.
- Все URL хранить только в `src/config`.
- Использовать `src/config/resources.json` как единый источник рабочих ссылок.
- Не создавать вымышленные ссылки.
- Поддерживать статусы ресурсов `ready`, `individual`, `missing`, `local`.
- Не добавлять backend без отдельного задания.
- Не добавлять форму авторизации в React.
- Реальная авторизация должна быть через Cloudflare Access перед статическим сайтом.
- Не хранить allowlist email в React как средство защиты.
- Не хранить пароли, токены и секреты.
- Не добавлять новые роли без отдельного задания.
- Не добавлять новые библиотеки без необходимости.
- Не выполнять деплой без отдельной команды пользователя.
- Не изменять другие проекты на компьютере.
- После изменений запускать `npm run typecheck`, `npm run validate:resources` и `npm run build`.
- Поддерживать мобильную версию.
- Интерфейс должен оставаться простым.
